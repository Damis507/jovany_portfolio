Td démenageur

cadre scolaire

La société DÉMGUIV(fictive), spécialisée dans le déménagement, a fait appel à LOGIMOUVE,
une entreprise de services du numérique, pour développer un logiciel de gestion des déménagements.
Le logiciel, appelé PSDEM, est développé en langage Java (et une version en php) et est composé de trois modules :
SUIVI_DEM, PLAN_DEM et GEST_ABS. La base de données Postgresql est située à Quimper et accessible
depuis toutes les agences de DÉMGUIV.

Outils et technologies : php et java, postgresql pour la base de donnée
